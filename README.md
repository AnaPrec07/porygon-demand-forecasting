# # Porygon Demand Forecasting

This repository contains code, data, and documentation for demand forecasting using machine learning models, with a focus on XGBoost. The project includes data collection, cleaning, model training, evaluation, and explainability.

## Project Structure

- **src/**  
  Main source code for the project.
  - **artifacts/models/xgboost/**: Saved XGBoost model artifacts, organized by timestamp.
  - **config/**: Configuration files and credentials.
  - **core/**: Core modules for configuration loading, clients (e.g., BigQuery), model definitions, and data transformations.
  - **notebooks/**: Jupyter notebooks for data exploration, model training, SHAP analysis and an inventory management simulation.
  - **pipelines/**: Training pipelines for XGBoost and other models.
  - **scripts/**: Shell scripts for environment setup and configuration.
